# Assets do NutriGest

Esta pasta contém os assets visuais do aplicativo.

## Arquivos Necessários

### ✅ Obrigatórios para Deploy:

1. **icon.png** (1024x1024px)
   - Ícone principal do app
   - Aparece nas stores e na tela inicial

2. **splash.png** (1242x2436px ou 1920x1080px)
   - Tela de carregamento inicial
   - Primeira impressão do usuário

3. **adaptive-icon.png** (1024x1024px)
   - Ícone adaptativo para Android
   - Área segura: 66% central

4. **favicon.png** (48x48px)
   - Ícone para versão web
   - Pode ser múltiplos tamanhos

## Como Criar

Veja o guia completo em `ASSETS_GUIDE.md` na mesma pasta.

### Opção Rápida:
1. Crie um design 1024x1024px
2. Use https://www.appicon.co/ para gerar todos os tamanhos
3. Coloque os arquivos aqui
